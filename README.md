# OpenClaw Skills Collection

个人定制的 OpenClaw 技能集合，涵盖美股、A股、加密货币分析。

## 技能列表

### 美股分析
| 技能 | 描述 |
|------|------|
| tech-earnings-deepdive | 美股财报深度分析（16模块+6视角框架）|

### 加密货币分析
| 技能 | 描述 |
|------|------|
| crypto-token-deepdive | 加密货币代币深度分析框架 |
| bitget-wallet-skill | Bitget钱包链上数据+swap |
| opennews | 6551加密新闻 |
| opentwitter | 6551 Twitter数据 |

### A股分析
| 技能 | 描述 |
|------|------|
| sector | A股板块分析 |
| a-share-tushare | Tushare Pro数据接口 |
| a-share-em | 东方财富API（无依赖）|
| a-share-finance | A股财务数据补充 |
| eastmoney-api | 东方财富接口文档 |
| a-share-deepdive | A股财报深度分析框架（16模块+6视角）|

## 安装方法

```bash
# 克隆到 OpenClaw skills 目录
git clone https://github.com/yourusername/openclaw-skills.git ~/.openclaw/workspace/skills
```

## 依赖

- Python 3.10+
- AKShare
- Tushare（需Token）
- 6551 API（需Token）
