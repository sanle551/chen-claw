# Finnhub Skill

Real-time financial data access.

## Usage

Set FINNHUB_API_KEY environment variable.

## Tools

- getQuote(symbol) - Get real-time stock quote
- getNews(symbol) - Get company news
- getFinancials(symbol) - Get financial statements
